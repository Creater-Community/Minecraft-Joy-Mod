# Minecraft-Joy-Mod
This is a Bedrock/Java Edition mod that adds new features or updates existing features while keeping the features of Minecraft.

## If you would like to help with this project
If you would like to help us with this project, please call us at the [Creator-Community](https://github.com/Creater-Community) discusstion and we will invite you when we are available!
